# To-Do List Application

## Structure
- `client/` - React frontend
- `server/` - Node.js/Express backend

## Setup

### Backend
```bash
cd server
npm install
npm start
```

### Frontend
```bash
cd client
npm install
npm start
```

### Run both together (from project root)
```bash
npm install          # once, to install concurrently
npm run dev          # starts server + client in sync
```
